# AI Music Generation

Create original, royalty-free music with AI.

## Overview

Gruvi's music generation creates complete songs with vocals, instruments, and production. All generated music is 100% copyright-free and can be used commercially.

## Generation Modes

### Standard Mode
- **Short**: 30-90 seconds (25 tokens)
- **Long**: 90-120 seconds (50 tokens)
- Quick generation
- Great for social content

### Premium Mode
- **Custom duration**: 30-180 seconds (50 tokens per 30 seconds)
- Higher quality
- More control over track length
- Better for music videos

## Writing Effective Prompts

### Good Prompt Structure
```
[Genre] song about [topic], [vocal type], [mood/energy], [instruments]
```

### Examples

**Pop Summer Hit**
```
An upbeat summer pop song about beach parties and good vibes,
female vocals, energetic and danceable, synths and guitars
```

**Emotional Ballad**
```
A heartfelt acoustic ballad about lost love and memories,
male vocals, emotional and intimate, piano and strings
```

**Product Jingle**
```
A catchy 30-second jingle for a coffee brand,
no vocals, upbeat and memorable, upbeat electronic
```

## Tips

1. **Be specific** - More detail = better results
2. **Include mood** - "energetic", "melancholic", "uplifting"
3. **Mention instruments** - Helps shape the sound
4. **Specify vocals** - Male, female, no vocals
5. **Reference style** - "like 80s synth-pop" works well
