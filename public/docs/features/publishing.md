# Social Media Publishing

Distribute your content to all major platforms from one place.

## Supported Platforms

| Platform | Post Types | Features |
|----------|------------|----------|
| YouTube | Videos, Shorts | Full metadata, thumbnails, scheduling |
| TikTok | Videos | Direct/draft posting, privacy settings |
| Instagram | Reels | Via Graph API, auto-formatting |
| Facebook | Videos, Posts | Page posting, scheduling |
| LinkedIn | Videos | Document-based sharing |
| X (Twitter) | Videos | Thread support |

## Connecting Accounts

### First-Time Setup

1. Go to **Account** â†’ **Connected Accounts**
2. Click **Connect** next to desired platform
3. Complete OAuth authorization

### Multiple Accounts

You can connect up to **3 accounts per platform**:
- Personal + Business accounts
- Multiple brand accounts
- Client accounts (agencies)

### Reconnecting

If connection expires:
1. Go to Connected Accounts
2. Click **Reconnect** on the affected account
3. Re-authorize

## Publishing Options

### Immediate Publish

Post content right now:
1. Select video from Library
2. Click **Publish**
3. Choose platforms
4. Add metadata
5. Click **Post Now**

### Scheduled Publish

Plan future posts:
1. Select video from Library
2. Click **Schedule**
3. Choose platforms
4. Add metadata
5. Set date and time
6. Click **Schedule**

### Draft Mode (TikTok)

Save to TikTok drafts for final review:
1. Select "Save as Draft" option
2. Content goes to your TikTok drafts
3. Open TikTok app to review and post

## Metadata

Each post can include:

### Title
- YouTube: Full title
- TikTok/Reels: Caption
- Platform-specific formatting

### Description
- Full description for YouTube/Facebook
- Caption for short-form platforms
- Hashtag support

### Hashtags
- AI-suggested hashtags
- Custom hashtags
- Platform-specific limits applied

### Thumbnail
Options:
1. **Auto-generate** - AI picks best frame
2. **Upload custom** - Your own image
3. **Select from scenes** - Choose specific video frame

**Token cost:** 10 tokens for AI thumbnail generation

## Best Practices

### Optimal Posting Times

| Platform | Best Times (General) |
|----------|---------------------|
| TikTok | 7-9am, 12-3pm, 7-11pm |
| Instagram | 11am-1pm, 7-9pm |
| YouTube | 2-4pm (Shorts), anytime (long-form) |
| LinkedIn | 7-8am, 12pm, 5-6pm (weekdays) |

*Note: Your audience may vary. Check analytics.*

### Cross-Platform Strategy

1. **Lead with TikTok/Reels** - Fastest feedback
2. **Repurpose to YouTube Shorts** - Same content, new audience
3. **LinkedIn for B2B** - Professional angle
4. **Facebook for reach** - Older demographics

### Content Adaptation

Same video, different approach:
- **TikTok**: Trendy music, casual caption
- **LinkedIn**: Professional insight, industry hashtags
- **YouTube**: SEO-optimized title and description
