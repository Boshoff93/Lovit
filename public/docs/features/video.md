# AI Video Generation

Create stunning AI-generated videos in multiple styles.

## Overview

Gruvi offers several video generation modes, each optimized for different content types:

| Type | Best For | Base Input |
|------|----------|------------|
| Cinematic | Story-driven content | Script/Prompt |
| Music Video | Song visualization | Audio track |
| Voiceover Video | Narrated content | Voiceover |
| UGC Creator | Product demos | Prompt + Voice |

## Token Cost

**50 tokens per second** of generated video

Example costs:
- 15-second video = 750 tokens
- 30-second video = 1,500 tokens
- 60-second video = 3,000 tokens

## Video Types

### Cinematic Stories

Create narrative-driven videos with AI-generated scenes.

**Best for:**
- Brand stories
- Product launches
- Emotional content
- Explainers

**How to use:**
1. Select "Cinematic" video type
2. Write your story/script
3. Choose visual style
4. Generate

### Music Videos

Visualize any song with matching AI-generated visuals.

**Best for:**
- Song promotion
- Lyric videos
- Visual albums
- Background content

**How to use:**
1. Select "Music Video" type
2. Choose or create a song
3. Describe visual theme
4. Generate

### Voiceover Videos

Combine narration with supporting visuals.

**Best for:**
- Tutorials
- Educational content
- Documentary style
- How-to videos

**How to use:**
1. Select "Voiceover Video" type
2. Choose or create voiceover
3. Describe visual style
4. Generate

## Visual Styles

Available styles include:
- **Realistic** - Photo-realistic generation
- **Cinematic** - Film-like color grading
- **Animated** - 2D/3D animation styles
- **Abstract** - Artistic interpretations
- **Product** - Clean commercial look

## Generation Process

1. **Storyboard** - AI creates scene breakdown
2. **Animation** - Each scene is generated
3. **Assembly** - Scenes are combined with audio
4. **Output** - Final video ready for download/publish

Typical generation time: 2-5 minutes depending on length.

## Tips for Better Videos

1. **Clear visuals in prompt** - Describe what you want to see
2. **Match audio and visuals** - Energetic music = dynamic visuals
3. **Consider platform** - Vertical for TikTok/Reels, horizontal for YouTube
4. **Use transitions** - Let the AI handle scene changes
5. **Test shorter first** - Start with 15s before committing to longer
