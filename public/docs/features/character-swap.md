# AI Character Swap

Replace people in videos with AI-generated characters.

## Overview

Character Swap uses the Kling AI model to replace people in existing videos with new characters. This is powerful for:
- Repurposing content with different faces
- Creating variations of successful videos
- Protecting privacy while keeping content
- Brand character integration

## Token Cost

**50 tokens per second** of processed video

## Swap Modes

### 1. Replace Character Only

Swap just the person while keeping the original environment.

**Best for:**
- Different avatar for same content
- A/B testing with different faces
- Localizing content for different markets

**How to use:**
1. Select "Replace Character" mode
2. Upload or select source video
3. Choose new character from AI Assets
4. Generate

### 2. Replace Character + Environment

Change both the person AND the background/setting.

**Best for:**
- Complete content transformation
- Putting product in new contexts
- Creating variations for different platforms

**How to use:**
1. Select "Replace Character + Environment" mode
2. Upload or select source video
3. Choose new character
4. Describe new environment
5. Generate

### 3. Custom Prompt Control

Full creative control with text prompts.

**Best for:**
- Specific creative visions
- Complex transformations
- Unique style applications

**Example prompt:**
```
Replace the person with a young professional woman
in a modern startup office environment,
maintain the same movements and expressions
```

## Reference Options

### Follow Video
The new character matches the movements and timing of the original video exactly.

### Follow Reference Image
Prioritize matching a specific character image, adapting it to the video movements.

**When to use each:**
- **Follow Video**: When movement accuracy is critical
- **Follow Reference**: When character consistency is critical

## Tips for Best Results

1. **Source video quality** - Higher quality in = better out
2. **Clear subject** - Person should be clearly visible
3. **Consistent lighting** - Avoid dramatic light changes
4. **Simple backgrounds** - Complex scenes are harder
5. **Smooth movements** - Avoid very fast motion
6. **Face visibility** - Partial face = partial results

## Use Cases

### Brand Mascot Videos
Create a consistent brand character across all content:
1. Generate your brand character as an AI Asset
2. Film or source base videos
3. Swap all videos to use your character
4. Build recognition with consistent face

Repurpose user testimonials or content without showing real faces:
1. Get original content from customers
2. Swap faces with AI characters
3. Keep authentic testimonials, protect privacy

Adapt content for different regions:
1. Create region-appropriate character assets
2. Swap characters to match local audience
