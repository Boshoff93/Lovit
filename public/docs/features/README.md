# Gruvi Features Guide

Complete documentation for all Gruvi capabilities.

## Content Creation

### [Music Generation](./music.md)
Create AI-generated songs in any genre or language.
- Standard vs Premium modes
- Short (30-90s) vs Long (90-180s) tracks
- 32 genres, 24 languages
- 100% royalty-free

### [Voiceover Creation](./voiceover.md)
Professional AI narration for your content.
- 25+ voice personalities
- Multiple languages
- UGC-style narration
- Voice change/cloning

### [Video Generation](./video.md)
Create stunning AI videos in multiple styles.
- Cinematic stories
- Music videos
- Voiceover videos
- UGC product demos

### [UGC Creator](./ugc-creator.md)
Generate influencer-style product videos.
- Prompt-based generation
- Voice change option
- Perfect for e-commerce
- Social-ready format

### [Character Swap](./character-swap.md)
Replace people in videos with AI characters.
- Replace character only
- Replace character + environment
- Custom prompt control
- Follow video or reference image

### [AI Assets](./ai-assets.md)

### [App Showcase](./app-showcase.md)

- Upload screenshots

- Perfect for app marketing

### [Media Upload](./media-upload.md)
Import your own content for remixing.
- Video upload
- Audio upload
- Use for character swaps
- Build custom assets
